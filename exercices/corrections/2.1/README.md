1. Les commandes à effectuer :
```bash
git config --global user.name 'glodie-tshimini'
git config --global user.email 'contact@tshimini.fr'
```
2. 
```bash
git config --help
```
- Ouvre la documentation de la commande `git config`
3. 
```bash
git config --list
```
- Voir la liste de paramètres qui peuvent être modifiés
4. 
```bash
git config --local help.format=web
```
- Modifier l'affichage de la documentation, par défaut une page web HTML

5. 
```bash
git init
```
- s'assurer au préalable d'être au bon endroit depuis votre terminal
