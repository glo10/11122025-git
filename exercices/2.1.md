# Exercice 2.1 : prise en main de git

1. Effectuez les configurations minimales à l'aide la commande `git config`
2. Utilisez la commande `git config --help` pour voir la documentation complète de cette commande
3. Effectuez la commande `git config --list` pour voir le résultat
- Pour quitter la liste affichée, appuyez sur ***la touche q***
4. Modifiez une variable de la configuration de Git de votre choix
5. Créez un nouveau dossier dans l'emplacement de votre choix et initialisez un nouveau projet Git dessus
6. Créez les dossiers admin/ et config/
7. Créez un fichier .gitignore à la racine de votre projet et ajoutez dans ce fichier les dossiers admin/ et config/
8. Créez un dossier img/ et créez un fichier .gitkeep à l'intérieur de ce dossier /img
