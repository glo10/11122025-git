# Correction J2 partie VI : GitHub

8. Question (Q) : vous créez un dépôt distant ***PUBLIC*** sur GitHub pour envoyer tout votre travail
- Réponse (R) : cf [vidéo création dépôt distant](../../../videos/create-new-repo.mp4)
9. Depuis GitHub, vous créez une nouvelle branche pour apporter une correction à votre menu
- R : cf. vidéo création d'une branche depuis GitHub
![branch](../../../videos/create_branch.mp4)
- Ici cette branche sera nommée *feature/menu-patch*
10. Vous récupérez cette nouvelle branche en local pour y ajouter vos corrections
```bash
git fetch origin
git checkout feature/menu-patch
```
