# Les commandes en cours

## Chapitre 1

1. git init
- Initialisation du projet Git
- Crée uniquement le dossier .git/
2. git add README.md exercices/2.1.md
- Ajout individuel de mes fichiers à versionner
- Désormais ses fichiers seront suivi par Git
3. git status : état du dépot
4. git commit -m "first commit" : mon premier commit (première version)
